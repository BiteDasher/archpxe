# archpxe
Quick setup of PXE server to load Arch Linux using an ethernet cable (boot is supported only in legacy mode)

## How to use?
Execute `sudo archpxe --help`

## Dependencies:
`bash` \
`coreutils` \
`grep` \
`dnsmasq` \
`darkhttpd` \
`curl` (for ISO download)

## AUR git clone link:
https://aur.archlinux.org/archpxe.git
